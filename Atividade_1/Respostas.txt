1-  Classe: Determina as propriedades (atributos) e comportamentos (métodos) que os objetos desse tipo terão. 
    As classes fornecem o molde a partir do qual os objetos são criados(Uma classe é um tipo a partir do qual objetos são criados).
    Objeto: É uma instância concreta de uma classe. 
    Representa uma entidade específica com estados (valores de atributos) e comportamentos (ações realizadas pelos métodos).


2-  Atributos: São as características de determinado objeto;
    Métodos: São os comportamentos desse objeto;
    Tendo como exemplo de objeto : Pessoa, os atributos poderiam ser: altura, peso, nacionalidade...; Os Métodos poderiam ser: Comprar, falar, andar e etc...
